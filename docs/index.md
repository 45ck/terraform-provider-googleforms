# Google Forms Provider

Manage Google Forms as infrastructure using Terraform.

This provider also includes optional Google Sheets and Google Drive resources to support common workflows like:

- placing forms/spreadsheets into Drive folders
- sharing documents via Drive permissions
- managing response spreadsheets (as spreadsheets) and writing bounded ranges
